# Лабораторная работа 3: Построение графиков функций

## Описание

Программа строит графики математических функций с использованием библиотек Matplotlib, NumPy и SymPy. Демонстрирует работу с символьными выражениями и их преобразование в вычислимые функции для визуализации.

## Основные компоненты

### Символьные выражения
**Назначение:** Определение функций в виде строковых формул.

**Используемые функции:**
- `func1_str = "0.98**x*sin(2*x)"` - первая функция (затухающая синусоида с частотой 2)
- `func2_str = "0.98**x*sin(5*x)"` - вторая функция (затухающая синусоида с частотой 5)

### Преобразование формул
**Процесс:**
1. `sympify()` - преобразует строку в символьное выражение SymPy
2. `lambdify()` - преобразует символьное выражение в вычислимую функцию NumPy

**Код:**
```python
func1 = sympify(func1_str)
func1_lambda = lambdify(x, func1, 'numpy')
```

### Генерация данных для графиков
**Функция:** `np.linspace(-10, 10, 400)`

**Параметры:**
- Начало диапазона: -10
- Конец диапазона: 10
- Количество точек: 400

**Результат:** Массив из 400 равномерно распределенных значений X

### Построение графиков
**Функция:** `plt.plot()`

**Параметры для первого графика:**
- `x_values, y1_values` - координаты точек
- `label` - метка для легенды
- `marker='o'` - маркеры в виде кружков
- `markersize=5` - размер маркеров
- `linestyle='-'` - сплошная линия
- `color='blue'` - синий цвет

**Параметры для второго графика:**
- `marker='s'` - маркеры в виде квадратов
- `linestyle='--'` - пунктирная линия
- `color='red'` - красный цвет

## Функции настройки графика

### `plt.title('Графики функций')`
**Назначение:** Устанавливает заголовок графика.

### `plt.xlabel('x')` и `plt.ylabel('y')`
**Назначение:** Подписывают оси координат.

### `plt.grid(True)`
**Назначение:** Включает отображение сетки для удобства чтения графика.

### `plt.legend()`
**Назначение:** Отображает легенду с метками функций.

### `plt.show()`
**Назначение:** Отображает окно с графиком.

## Особенности реализации

### Символьные вычисления
- Использование SymPy позволяет работать с формулами в символьном виде
- Преобразование в NumPy функции обеспечивает быстрые вычисления для больших массивов

### Визуализация
- Два графика на одном рисунке для сравнения
- Разные стили линий и маркеров для различия функций
- Автоматическое масштабирование осей

### Математические функции
- **0.98**x** - экспоненциальное затухание (амплитуда уменьшается)
- **sin(2*x)** и **sin(5*x)** - синусоидальные колебания с разными частотами

## Результат

Программа отображает окно с графиком, на котором видны две затухающие синусоиды:

Обе функции демонстрируют экспоненциальное затухание амплитуды при удалении от начала координат.
