# Лабораторная работа 2: Простые фигуры в OpenGL

## Описание

Программа демонстрирует работу с библиотекой OpenGL через GLFW для отрисовки простых геометрических фигур: треугольника, квадрата и круга. Используется ортогональная проекция и цветные вершины для создания градиентных эффектов.

## Основные функции

### `clear_screen()`
**Назначение:** Очищает буферы цвета и глубины перед новым кадром.

**Использование:** Вызывается в начале каждого цикла отрисовки для подготовки экрана.

### `draw_triangle(vertices, colors)`
**Назначение:** Рисует треугольник с цветными вершинами.

**Параметры:**
- `vertices` (list) - список из 3 вершин, каждая вершина - кортеж (x, y, z)
- `colors` (list) - список из 3 цветов, каждый цвет - кортеж (R, G, B) в диапазоне 0.0-1.0

**Особенности:** Создает градиентную заливку между вершинами благодаря интерполяции цветов OpenGL.

### `draw_quad(vertices, colors)`
**Назначение:** Рисует четырехугольник (квадрат) с цветными вершинами.

**Параметры:**
- `vertices` (list) - список из 4 вершин
- `colors` (list) - список из 4 цветов для каждой вершины

**Особенности:** Использует примитив `GL_QUADS` для отрисовки четырехугольника.

### `draw_polygon(center, radius, num_sides, color=None, mode=GL_POLYGON)`
**Назначение:** Рисует правильный многоугольник (или круг при большом количестве сторон).

**Параметры:**
- `center` (tuple) - центр многоугольника (x, y, z)
- `radius` (float) - радиус описанной окружности
- `num_sides` (int) - количество сторон многоугольника
- `color` (tuple/list) - цвет заливки или список цветов для каждой вершины
- `mode` (GLenum) - режим отрисовки (GL_POLYGON, GL_LINE_LOOP и т.д.)

**Особенности:** 
- Использует `numpy.linspace` для равномерного распределения вершин по окружности
- При большом количестве сторон (50+) создает эффект круга
- Поддерживает как одноцветную, так и многоцветную заливку

## Структура программы

### Инициализация
1. Инициализация GLFW
2. Создание окна 800x600 пикселей
3. Установка ортогональной проекции (`glOrtho(-1, 1, -1, 1, -1, 1)`)

### Основной цикл отрисовки
1. Очистка экрана
2. Отрисовка треугольника с градиентом (желтый-красный-голубой)
3. Отрисовка квадрата с четырьмя цветами
4. Отрисовка круга (50-угольник) серого цвета
5. Обновление буферов

## Как запустить

1. Установите зависимости:
   ```bash
   pip install glfw PyOpenGL numpy
   ```
2. Запустите программу:
   ```bash
   python lab2.py
   ```

## Зависимости

- `glfw` - библиотека для создания окон и обработки событий
- `OpenGL.GL` - основные функции OpenGL
- `OpenGL.GLU` - утилиты OpenGL
- `numpy` - для математических операций с массивами

## Особенности реализации

- **Ортогональная проекция:** Используется `glOrtho` для простой 2D-проекции без перспективы
- **Цветные вершины:** Каждая вершина имеет свой цвет, OpenGL автоматически интерполирует цвета между вершинами
- **Примитивы OpenGL:** Используются `GL_TRIANGLES`, `GL_QUADS` и `GL_POLYGON` для разных фигур
- **Непрерывный рендеринг:** Программа работает в цикле, постоянно перерисовывая сцену

## Примеры фигур

- **Треугольник:** Вершины в углах экрана с градиентной заливкой
- **Квадрат:** Центральный квадрат с четырьмя разными цветами вершин
- **Круг:** 50-угольник, визуально похожий на круг, серого цвета
