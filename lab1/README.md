# Лабораторная работа 1: Базовые графические примитивы

## Описание

Программа демонстрирует работу с базовыми графическими примитивами в Python с использованием библиотеки Tkinter. Реализованы классы для работы с линиями, эллипсами и многоугольниками с возможностью их отрисовки и стирания.

## Основные компоненты

### Класс `Pen`
**Назначение:** Представляет инструмент для рисования (кисть/перо).

**Параметры:**
- `thickness` (int) - толщина линии
- `color` (str) - цвет в формате HEX (например, '#FF0000')

**Использование:**
```python
pen = Pen(thickness=2, color='#FF0000')
```

### Класс `Line`
**Назначение:** Представляет линию с различными стилями.

**Параметры:**
- `start_point` (Tuple[int, int]) - начальная точка линии
- `end_point` (Tuple[int, int]) - конечная точка линии
- `pen` (Pen) - инструмент для рисования
- `line_type` (str) - тип линии: 'solid' (сплошная) или 'dash' (пунктирная)
- `cap_style` (str) - стиль концов линии: 'round' (округлые)

**Методы:**
- `draw(canvas)` - отрисовывает линию на холсте
- `erase(canvas, background_color)` - стирает линию, перерисовывая её цветом фона

### Класс `Ellipse`
**Назначение:** Представляет эллипс (или круг).

**Параметры:**
- `center` (Tuple[int, int]) - центр эллипса
- `radius_x` (float) - радиус по оси X
- `radius_y` (float) - радиус по оси Y
- `outline_color` (str) - цвет контура
- `fill_color` (str) - цвет заливки

**Методы:**
- `draw(canvas)` - отрисовывает эллипс на холсте
- `label(canvas, label_text, font_size)` - добавляет текстовую метку в центр эллипса
- `erase(canvas, background_color)` - стирает эллипс

### Класс `Polygon`
**Назначение:** Представляет многоугольник произвольной формы.

**Параметры:**
- `vertices` (List[Tuple[int, int]]) - список вершин многоугольника
- `outline_color` (str) - цвет контура
- `fill_color` (str) - цвет заливки

**Статический метод:**
- `create_regular_polygon(center, num_sides, side_length)` - создает правильный многоугольник с заданным количеством сторон

**Методы:**
- `draw(canvas)` - отрисовывает многоугольник на холсте
- `erase(canvas, background_color)` - стирает многоугольник

## Функции интерфейса

### `draw_all()`
**Назначение:** Отображает все созданные фигуры на холсте.

**Использование:** Вызывается при нажатии кнопки "Draw All Shapes".

### `clear_canvas()`
**Назначение:** Очищает весь холст, удаляя все нарисованные объекты.

**Использование:** Вызывается при нажатии кнопки "Clear Canvas".

## Пример использования

Программа создает окно с холстом и двумя кнопками:
- **Draw All Shapes** - рисует примеры линий, эллипсов и многоугольников
- **Clear Canvas** - очищает холст

По умолчанию создаются:
- Пунктирная красная линия
- Желтый эллипс с синим контуром
- Серый треугольник с зеленым контуром
