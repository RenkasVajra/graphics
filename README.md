# Лабораторные работы по компьютерной графике

Коллекция лабораторных работ по компьютерной графике, реализованных на Python. Каждая работа демонстрирует различные аспекты работы с графикой: от простых примитивов до сложных алгоритмов трассировки лучей.

## Структура проекта

```
графика/
├── lab1/          # Базовые графические примитивы (Tkinter)
├── lab2/          # Простые фигуры в OpenGL
├── lab3/          # Построение графиков функций (Matplotlib)
├── lab4/          # Треугольник Серпинского (фракталы)
├── lab5/          # Обработка изображений
├── lab6/          # Трехмерные объекты в OpenGL
├── lab7/          # Моделирование робота в OpenGL
└── lab8/          # Алгоритм обратной трассировки лучей
```

## Описание лабораторных работ

### Lab 1: Базовые графические примитивы
Реализация классов для работы с линиями, эллипсами и многоугольниками в Tkinter. Демонстрирует объектно-ориентированный подход к работе с графикой.

**Технологии:** Tkinter, Python

### Lab 2: Простые фигуры в OpenGL
Отрисовка треугольника, квадрата и круга с использованием OpenGL и GLFW. Показывает работу с цветными вершинами и градиентной заливкой.

**Технологии:** OpenGL, GLFW, NumPy

### Lab 3: Построение графиков функций
Визуализация математических функций с использованием Matplotlib и SymPy. Демонстрирует символьные вычисления и построение графиков.

**Технологии:** Matplotlib, NumPy, SymPy

### Lab 4: Треугольник Серпинского
Реализация рекурсивного алгоритма построения фрактала "Треугольник Серпинского". Показывает работу с рекурсивными алгоритмами в графике.

**Технологии:** OpenGL, GLFW, NumPy

### Lab 5: Обработка изображений
Графический редактор для загрузки, обработки и анализа изображений. Реализована инверсия цветов и отображение данных о пикселях.

**Технологии:** Tkinter, PIL (Pillow)

### Lab 6: Трехмерные объекты в OpenGL
Интерактивная 3D-сцена с кубом, спиралью и додекаэдром. Поддержка вращения с помощью мыши и перспективной проекции.

**Технологии:** Pygame, OpenGL, NumPy

### Lab 7: Моделирование робота в OpenGL
Создание трехмерной модели робота из простых геометрических примитивов. Интерактивное управление вращением с клавиатуры.

**Технологии:** OpenGL, GLFW, NumPy

### Lab 8: Алгоритм обратной трассировки лучей
Полнофункциональная реализация алгоритма трассировки лучей с поддержкой отражений, преломлений, теней и GUI для управления параметрами.

**Технологии:** NumPy, PIL, Tkinter, Threading

## Установка

1. Клонируйте репозиторий:
```bash
git clone <repository-url>
cd графика
```

2. Создайте виртуальное окружение (рекомендуется):
```bash
python -m venv venv
source venv/bin/activate  # Linux/Mac
# или
venv\Scripts\activate  # Windows
```

3. Установите зависимости:
```bash
pip install numpy glfw PyOpenGL matplotlib sympy pillow pygame
```

## Особенности реализации

### Графические библиотеки
- **Tkinter** - для простых 2D-примитивов и GUI
- **OpenGL** - для 3D-графики и сложных визуализаций
- **Matplotlib** - для построения графиков функций
- **PIL/Pillow** - для обработки изображений

### Алгоритмы
- Рекурсивные алгоритмы (фракталы)
- Трассировка лучей с рекурсивными отражениями и преломлениями
- Математические модели освещения (Фонг, Ламберт)
- Геометрические вычисления (пересечения, нормали, трансформации)

### Интерактивность
- Управление с клавиатуры (Lab 7)
- Управление мышью (Lab 6)
- GUI с слайдерами и кнопками (Lab 1, Lab 5, Lab 8)
